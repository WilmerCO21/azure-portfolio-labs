# Azure Portfolio Labs (Hands-On)

Hands-on Azure labs built to demonstrate practical cloud skills for an Azure/Cloud Intern role.

## Labs Included
- **Lab 01 — VM + NSG + SSH + Nginx**
  - Resource Group, VNet/Subnet, NSG (SSH restricted), Ubuntu VM
  - SSH connection and Nginx installation
  - HTTP access validation (port 80)

- **Lab 02 — Azure Storage (Blob)**
  - Storage Account creation
  - Blob container creation
  - Upload/download files and access testing

- **Lab 03 — Monitoring + Alerts**
  - Azure Monitor alert rule (CPU threshold)
  - Action Group (email notification)
  - Alert trigger test using CPU stress tool

- **Lab 04 — RBAC (IAM)**
  - Role assignment at Resource Group scope
  - Verification of role assignments and permissions

## Tech Stack
- Microsoft Azure (Portal)
- Ubuntu Linux
- SSH
- Nginx
- Azure Storage (Blob)
- Azure Monitor (Alerts + Action Groups)
- RBAC / IAM
